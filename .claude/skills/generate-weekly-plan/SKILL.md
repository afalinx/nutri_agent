# Генерация плана питания на неделю

## Описание
Генерирует план питания на 7 дней, составляя каждый день по отдельности с учётом разнообразия. После генерации всех дней — формирует агрегированный список покупок.

## Когда использовать
- "Составь план на неделю"
- "Меню на 7 дней"
- "Рацион на неделю + список покупок"

## Инструкции

### Шаг 1: Контекст
```bash
cd backend && PYTHONPATH=. python cli.py context --user-id <USER_ID> --day 1
```

### Шаг 2: Генерация 7 дней (автономно)
Для каждого дня (1-7) составь план, используя навык `generate-meal-plan` с обязательным циклом `validate -> auto-fix` до `valid=true`.
**Важно для разнообразия:**
- Не повторяй одни и те же блюда в соседних днях
- Чередуй типы белка (курица → рыба → говядина → тофу)
- Каждый день должен попадать в ±5% от целевого калоража

Собери все 7 дней в единый JSON:
```json
{
  "total_days": 7,
  "daily_target_calories": <int>,
  "days": [
    { "day_number": 1, "total_calories": ..., "meals": [...] },
    { "day_number": 2, ... },
    ...
  ]
}
```

### Шаг 3: Список покупок
```bash
cd backend && PYTHONPATH=. python cli.py shopping-list --file /tmp/weekly_plan.json
```

### Шаг 4: Сохранение
```bash
cd backend && PYTHONPATH=. python cli.py save --user-id <USER_ID> --file /tmp/weekly_plan.json --days 7
```

### Шаг 5: Результат
Покажи пользователю:
1. Таблицу плана (день → блюда → КБЖУ)
2. Итого КБЖУ по каждому дню
3. Агрегированный список покупок
4. `plan_id` после сохранения
